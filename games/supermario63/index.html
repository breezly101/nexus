Super Mario 63 Redux [Tiny Demo]

Note, due to some camera issues, some jittering effects may occur in certain parts of the game. Discretion is advised.

Welcome to Super Mario 63 Redux!

This is the Tiny Demo, consisting of the title screen, main menu, and Tutorial 1. To help us improve the game, we encourage you to report bugs and issues using the feedback menu, as this sends the data directly to the team. Feel free to use this for things you like about the game, too! It helps to know what we 're doing right as well as what we 're doing wrong.

You can join the Discord server here: https://discord.gg/kmQVVGQFWn
Subscribe to our YouTube: https://www.youtube.com/@ReduxTeam
Follow us on X here: https://x.com/TeamRedux
Follow us on BlueSky here: https://bsky.app/profile/reduxteam.bsky.social
Check out our website here: https://sm63redux.com

Controls:

  • Arrow keys - Movement
  • Z - Interact, ground pound
  • X - Spin
  • C - Use FLUDD
  • P - Pause game
  • F1 / Top left button on mobile  - Open feedback menu

Controls that currently don 't work on mobile:

  • A - Reset
  • = - Zoom in
  • - - Zoom out

Install instructions

The MacOS version is not signed, so you should run this command to fix it:

xattr -rds com.apple.quarantine [drag the app here]

The Linux version is not marked as executable, so you should run this Bash command to fix it:

chmod +x "Super Mario 63 Redux.x86_64 "
